# Game Proposal
## Game overview
- *Metroid*vania, sci-fi
- Findable required powerups
- Hollow knight style save points
- Side scroller simple platforming with puzzles
- Support gamepad
### Target Players
Tommy (Super metroid fans)

## Core Mechanics and Features
- Side scroller shooting, like in super metroid
- Aim direction is controlled by right stick or mouse (twin stick shooter-ish style, similar to terraria cursor)
- Movement upgrades
    - Dash
    - Double jump
    - wall jump
    - jetpack / wings (terraria wings kinda)
- State machine based AI controllers
- save points (static spots, when you die level progress is saved but you return to that save point
- Levels / Themed areas

## Tasks
### Core
- Basic user HUD
- Generic node based state machine
- Shooting effects (particle effects)
- Player controller
- Enemy AI
- Level Design
- Powerups (node based)
- Collectables
- Save points / checkpoints
- 4 themed areas
### Stretch
- Randomization (archipelago integration for really far stretch goal)
- More areas
- end boss
- Easter eggs

## Scope and Complexity
- Game has modular design, challenging to implement
- Tasks are separated into core goals and stretch goals, there is plenty to do in stretch goals (primarily being randomization)

## New Knowledge / Open Questions
- Save states
- Player animation
- Intractable level elements (levels/buttons)
- Particle systems
- Node based power-ups
- Possible rule based randomization
- Parallax background
